js 中数组对象某一索引求和

2023-01-05 1 min read # javascript

之前做一个数组对象求和功能时用的 forEach 感觉不够简单。
后来在网上找到了内置的 reduce 函数刚好可以解决这个求和问题

let datas = [
        {a: 15, b: 2},
        {a: 23, b: 1},
        {a: 45, b: 8},
        {a: 66, b: 7},
        {a: 33, b: 3}
    ];
    // 输出 182
    console.log(datas.reduce((total, item) => total + item.a, 0));
    ```
